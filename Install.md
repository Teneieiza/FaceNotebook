## Download .Net Core
https://dotnet.microsoft.com/en-us/download

## Install Extention
C# Dev Kit

## Create Project
1. Open cmd
2. cd to folder project
3. dotnet new webapi -n (Name)

## Run in Terminal
1. dotnet add package Swashbuckle.AspNetCore
2. dotnet run
